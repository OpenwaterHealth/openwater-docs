# Make Your First Contribution

Ready to contribute? Great! Follow these steps:

## 1. Find an Issue

Look for issues labeled `good first issue`:

- [OpenLIFU Issues](https://github.com/OpenwaterHealth/OpenLIFU-python/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
- [OpenMOTION Issues](https://github.com/OpenwaterHealth/OpenMOTION-platform/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)

## 2. Fork and Clone

```bash
# Fork on GitHub, then:
git clone https://github.com/YOUR-USERNAME/REPO-NAME.git
```

## 3. Make Changes

Create a branch, make your changes, commit with clear messages.

## 4. Submit PR

Push your changes and open a Pull Request!
