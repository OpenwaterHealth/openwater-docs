# System Requirements

## Software Requirements

- Python 3.9 or higher
- pip 21.0 or higher
- Git 2.30 or higher

## Hardware Requirements

- 4GB RAM minimum (8GB recommended)
- 2GB free disk space
- USB port for hardware connectivity (optional)

## Supported Platforms

- Linux (Ubuntu 20.04+, RHEL 8+)
- macOS 11+ (Big Sur and later)
- Windows 10/11
