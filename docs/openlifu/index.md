# OpenLIFU Platform

**Low Intensity Focused Ultrasound for Neuromodulation**

OpenLIFU is an open-source platform for planning and delivering focused ultrasound treatments.

## Quick Start

```bash
pip install openlifu
```

## Features

- Treatment planning
- Acoustic simulation
- Hardware control
- 3D Slicer integration
